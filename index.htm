ngSlide

This package provides a method to write a presentation with a sinlge html file.

The html-document is structured as followed:

			<presentation>
				<slide>

				</slide>
				<slide>

				</slide>
			</presentation>

Controller

With the angularjs controllers there are control-buttons and -keys provided to cicle throught the slides.

Also the number of the actual slide and the number of all slides are displayed.

Slide Numbers

In the upper right corner there is the number of the actual slide
and separated by / the number of all slides.
